# icehost

必填变量
| 变量名                  | 说明                                |
| -------------------- | --------------------------------- |
| `ICEHOST_SERVER_URL` | IceHost 服务器管理页面地址                 |
| `ICEHOST_COOKIES`    | 从浏览器导出的 IceHost 登录 Cookie，JSON 格式 |

可选变量
| 变量名            | 说明                     |
| -------------- | ---------------------- |
| `TG_BOT_TOKEN` | Telegram Bot Token     |
| `TG_CHAT_ID`   | Telegram 接收消息的 Chat ID |
